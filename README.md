# System-rozpoznawania-mowy
System rozpoznawania mowy do projektu inżynierskiego

Aby system działał, należy: 
- pobrać model dostępny pod adresem: https://alphacephei.com/vosk/models/vosk-model-small-pl-0.22.zip
- wypakować go w folderze, w którym znajduje się plik .py
- zmienić nazwę folderu z modelem na: model
